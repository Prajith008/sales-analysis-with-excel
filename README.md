# sales-analysis-with-excel

This project is a Dynamic Sales Dashboard created in Microsoft Excel to provide an interactive and comprehensive view of sales performance. It includes dynamic charts, graphs, and key performance indicators (KPIs) that help users track total sales, profits, and growth trends in real-time. With user-friendly filters and slicers, the dashboard allows anyone to easily analyze sales data by region, product, or time period, making it a practical tool for businesses, students, or professionals who want to present and understand sales insights clearly and efficiently.
